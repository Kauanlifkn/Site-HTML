O que é HTML?

Voltar

O HTML é uma linguagem de marcação de texto. Isso mesmo ela não é uma linguagem de programação, como muitos contumam pensar por ai. Mas isso não quer dizer que ela deixe de ser importante para o mundo tecnologico de forma geral, pelo contrario. É uma tecnolgoia ultilizada em muitos lugares, para a produção de sites e até app (mas isso já é uma coisa mais complexa).

O que eu preciso saber para me tornar um desenvolvedor HTML?

Voltar

Existem muitos assuntos necessarios para que uma pessoa possa se tornar um bom desenvolvedor HTML, mas entre eles estão:

  • Conhecer sobre Banco de dados

Banco de dados é um conhecimento essencial para quem quer entrar de cabeça nesse mercado. Entender a relação entre servidor e cliente e sem duvidas um conhecimento que se adquirido de uma forma completa, vai ajudar muito na sua carreira como desenolvedor web.

  • Software x Hardwares

Conhecer a diferença entre os dois é crucial para um bom entendimento sobre o assunto. Entender que Software é o que xingamos e Hardwares é o que batemos é necessario nessa carreira.

  • Comandos

Depois de tudo isso, é importante estudar os comandos bases do HTML, para que assim você possa se desenolver na pratica e entender mais sobre o assunto.
